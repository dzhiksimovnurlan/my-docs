# Phone Cleaner Plus

Веб-приложение для подписки на сервис очистки телефона с интеграцией Stripe для обработки платежей.

## Возможности

- Подписка с пробным периодом
- Оплата картой, Apple Pay, Google Pay
- FastAPI бэкенд + статический фронтенд
